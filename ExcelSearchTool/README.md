# 📊 ExcelSearchTool

> 폴더 내 다수의 엑셀 파일을 한 번에 키워드 검색하고, 결과에 따라 파일을 자동 정리하는 WinForms 도구

여러 엑셀 파일을 하나씩 열어 "Ctrl+F"로 검색하던 수동 작업을 자동화하기 위해 개발된 도구입니다.
폴더 안의 `.xls`/`.xlsx` 파일을 일괄 검색하고, 검색어 포함 파일을 지정 폴더로 이동까지 한 번에 처리합니다.

---

## 🛠 기술 스택

- C# / WinForms (.NET Framework 4.7.2~4.8)
- NPOI (엑셀 .xls / .xlsx 읽기)
- Properties.Settings (사용자별 설정 저장)

---

## 🏗 System Architecture

```text
[작업 폴더] 엑셀 파일들 (.xls / .xlsx)
    ↓ (폴더 선택 → 파일 목록 자동 표시)
[CheckedListBox] 검사 대상 선택 (전체/개별)
    ↓ (검색어 입력 → 검색 실행)
[NPOI] 전체 시트 / 전체 셀 순회 → 키워드 매칭
    ↓
[DataGridView] No / 파일명 / 매칭개수 표시
    ↓ (이동 버튼 클릭 → 확인창)
[이동 폴더] 검색어 포함 파일 이동
```

## 📂 화면 구성

### 메인 화면

| 영역 | 컨트롤 | 설명 |
|---|---|---|
| 상단 | 폴더 선택 버튼 / 경로 라벨 | 폴더 선택 시 파일 목록 자동 로드 (ToolTip으로 전체 경로 확인) |
| 상단 | 설정 버튼 | 설정 화면 열기 (메인 폼 우측에 표시) |
| 목록 | CheckedListBox + 전체선택 체크박스 | 검사 대상 파일 선택 |
| 검색 | 검색어 입력 + 검색 버튼 | 부분일치 검색 실행 |
| 결과 | DataGridView (No/파일명/매칭개수) | 파일별 매칭 결과 표시 |
| 이동 | 이동 버튼 | 검색어 포함 파일을 설정된 폴더로 이동 (확인창 포함) |

### 설정 화면

| 항목 | 설명 |
|---|---|
| 기본 폴더 경로 | 실행 시 자동으로 불러올 폴더 (폴더 선택 + 저장) |
| 이동 폴더 경로 | 검색어 포함 파일을 옮길 폴더, 완전 별도 경로 (폴더 선택 + 저장) |
| 대소문자 구분 | 구분함 / 구분안함 (라디오, 기본값 구분안함) |
| 숨김 시트 포함 | 포함함 / 포함안함 (라디오, 기본값 포함안함) |
| 초기화 | 모든 설정을 "지정안됨"/기본값으로 되돌림 |

## 🚀 사용 방법

### 자동 실행 흐름

```text
프로그램 실행
    ↓
설정에 저장된 기본 폴더 자동 로드 (있으면)
    ↓
파일 목록 표시 (기본 전체선택)
    ↓
검사 대상 선택 → 검색어 입력 → 검색 실행
    ↓
DataGridView에 파일별 매칭개수 표시
    ↓
이동 버튼 → 확인창("N개 파일을 [경로]로 이동하시겠습니까?")
    ↓
검색어 포함 파일 → 이동 폴더로 이동, 목록 자동 갱신
```

### 단계별 사용법
1. 폴더 선택 (또는 설정에서 지정한 기본 폴더 자동 로드)
2. 검사할 파일 체크 (기본 전체선택, 개별 해제 시 전체선택 자동 해제)
3. 검색어 입력 후 "검색" 클릭
4. 결과 표에서 파일별 매칭개수 확인
5. "이동" 클릭 → 확인창에서 개수/경로 확인 후 진행

## 🧩 검색 규칙

| 항목 | 동작 |
|---|---|
| 검색 방식 | 부분일치 (`LIKE '%keyword%'`, 엑셀 "모두 찾기"와 동일) |
| 검색 대상 | 모든 시트의 모든 셀 값 |
| 대소문자 구분 | 설정값 기준 (기본: 구분안함) |
| 숨김 시트 | 설정값 기준 (기본: 포함안함, `IsSheetHidden`/`IsSheetVeryHidden`) |
| 파일 사용 중 | 알림창 표시 + 결과는 "사용 중"으로 표시 |

> **실패 우선이 아닌 단순 매칭 원칙**: 셀 값에 키워드가 포함되면 그대로 카운트합니다.
> 수식 셀은 계산된 결과값(`cell.ToString()`) 기준으로 비교합니다.
